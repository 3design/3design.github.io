# the BigBrain

a high-resolution brain atlas and dataset

The BigBrain is an ultra-high resolution brain atlas and dataset. At 20 microns resolution, the total dataset is over a terabyte in size and is open to the public.

The BigBrain can be experienced in a browser using Neuroglancer. Alternately, you can access the dataset itself in LORIS.

This website is currently under construction and more content will follow shortly.
